# BGLUGwatch info  
## What is BGLUG?  
BGLUG is a User's Group that focuses on Linux. Every month we gather and we also have a mailing list.  
## So what is BGLUGwatch?  
BGLUGwatch is basically a program written in Python to view most of [the website](http://bglug.ca)

Yes, I know, but you gotta start somewhere.
